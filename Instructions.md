## ✨ Key Features
- **Dynamic Sliding Panel**: Uses Framer Motion's spring physics for a snappy feel.
- **Enhanced Hovers**: 3D lifts on inputs and glowing borders on social icons.
- **Fully Responsive**: Wide split-view on Desktop and stacked view on Mobile.
- **Modern Tech**: Built with Next.js, Tailwind CSS, and Lucide Icons.

## 🛠️ Installation & Setup
To get this component running, install the following dependencies:

npm install framer-motion lucide-react clsx tailwind-merge

⚙️ Utility File
Ensure your src/lib/utils.js contains the cn helper:
import { clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs) {
  return twMerge(clsx(inputs));
}
